# Режим AUTOCOMMIT

```r
    MySQL по умолчанию работает в режиме AUTOCOMMIT. Это означает, что если вы не начали транзакцию явным образом, 
    каждый запрос автоматически выполняется в отдельной транзакции. Вы можете включить или отключить режим AUTOCOMMIT для текущего соединения
```
```sql 
    SHOW VARIABLES LIKE 'AUTOCOMMIT'; 
```
